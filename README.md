# note フォロワーチェッカー

note.com のフォロワーの増減を確認できる Chrome 拡張機能です。

## 機能

- フォロワーリストを取得し、前回との差分（新規・フォロー外し）を表示
- バックグラウンドで取得するため、途中でポップアップを閉じても処理が継続
- 10分のクールダウンで連続実行を防止

## インストール

1. このリポジトリを ZIP でダウンロードして展開
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」から展開したフォルダを選択

## 使い方

1. 拡張機能アイコンをクリックしてポップアップを開く
2. 右下の「設定」から自分の note urlname を入力・保存
   - 例: `https://note.com/your_urlname` なら `your_urlname`
3. 「チェック」ボタンを押す
4. 初回はベースライン（比較基準）として保存されます
5. 2回目以降から新規フォロワーとフォロー外しが表示されます

## 注意事項

- note.com の非公式 API を使用しているため、仕様変更により動作しなくなる場合があります
- フォロワーが 1000 人を超える場合、API の制限により全件取得できないことがあります（警告表示あり）
